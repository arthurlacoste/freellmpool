Free Groq API

Groq offers a free API tier known for very fast inference. You get a free key at console.groq.com/keys, the endpoint is OpenAI-compatible at https://api.groq.com/openai/v1, and free models include Llama 3.3 70B, Llama 4 Scout, and gpt-oss 20B/120B. Free limits are roughly ~1,000 requests/day per model (check current docs). To go beyond Groq's cap, pool it with other free tiers using freellmpool, which fails over automatically when Groq is rate-limited.

Use Groq's free API directly

curl https://api.groq.com/openai/v1/chat/completions \
  -H "Authorization: Bearer $GROQ_API_KEY" -H "Content-Type: application/json" \
  -d '{"model":"llama-3.3-70b-versatile","messages":[{"role":"user","content":"Hi"}]}'

Free models on the tier: llama-3.3-70b-versatile, llama-3.1-8b-instant, openai/gpt-oss-120b, openai/gpt-oss-20b, meta-llama/llama-4-scout-17b-16e-instruct, and more.

Pool Groq with other free tiers (one endpoint, failover)

Groq's free cap is generous but finite. freellmpool pools Groq with Cerebras, Gemini, NVIDIA and a dozen others so a 429 on Groq transparently falls through to the next provider:

pip install freellmpool
export GROQ_API_KEY=...          # plus any other free keys you have
freellmpool ask -p groq "Write a haiku about sqlite"   # pin Groq
freellmpool ask "..."                                  # or let it pool + fail over

Run a proxy (freellmpool proxy) and your existing OpenAI code uses Groq + the pool unchanged. See using multiple free LLM APIs together.

FAQ

Is the Groq API free?

Yes, Groq has a free tier. Create a key at console.groq.com/keys and call the OpenAI-compatible endpoint at api.groq.com/openai/v1. Free usage is rate-limited (roughly ~1,000 requests/day per model; verify current limits).

What models are free on Groq?

Free Groq models include Llama 3.3 70B Versatile, Llama 3.1 8B Instant, Llama 4 Scout, and OpenAI's gpt-oss 20B/120B, among others.
