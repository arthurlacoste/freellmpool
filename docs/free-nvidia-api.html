Free NVIDIA NIM API

NVIDIA's build.nvidia.com gives free API access to a wide catalog of models via NIM. Get a free key at build.nvidia.com; the endpoint is OpenAI-compatible at https://integrate.api.nvidia.com/v1 and covers dozens of free models including Llama 3.3 70B, NVIDIA Nemotron, Kimi K2, and Mistral Small. Free usage is rate-limited via credits. Pool NVIDIA with other free tiers using freellmpool for failover and combined capacity.

Use NVIDIA's free API directly

curl https://integrate.api.nvidia.com/v1/chat/completions \
  -H "Authorization: Bearer $NVIDIA_API_KEY" -H "Content-Type: application/json" \
  -d '{"model":"meta/llama-3.3-70b-instruct","messages":[{"role":"user","content":"Hi"}]}'

A few of the free models: meta/llama-3.3-70b-instruct, moonshotai/kimi-k2.6, nvidia/nemotron-3-super-120b-a12b, mistralai/mistral-small-4-119b-2603 — dozens in total.

Pool NVIDIA with other free tiers

NVIDIA's breadth pairs well with faster tiers like Groq and Cerebras. freellmpool lets you use them together and fail over automatically:

pip install freellmpool
export NVIDIA_API_KEY=...         # plus other free keys
freellmpool models -p nvidia      # list NVIDIA's free models
freellmpool ask "..."             # pool + fail over

FAQ

Is the NVIDIA NIM API free?

Yes. build.nvidia.com offers free API access (rate-limited via credits) to a large catalog of models. Create a key at build.nvidia.com and call the OpenAI-compatible endpoint at integrate.api.nvidia.com/v1.

What models can I use free on NVIDIA?

The free catalog is broad — Llama 3.3 70B, NVIDIA Nemotron family, Kimi K2, Mistral Small and many more, all through one OpenAI-compatible endpoint.
