Free Gemini API

Google offers a free Gemini API tier through Google AI Studio. Get a free key at aistudio.google.com/apikey; free models include Gemini 2.5 Flash, 2.0 Flash, and 2.0 Flash-Lite. The free tier allows roughly ~1,500 requests/day on Flash models (verify current limits). To combine Gemini's free quota with other providers and fail over when it's exhausted, pool it with freellmpool.

Use Gemini's free API directly

Gemini has both a native API and an OpenAI-compatible endpoint:

curl "https://generativelanguage.googleapis.com/v1beta/openai/chat/completions" \
  -H "Authorization: Bearer $GEMINI_API_KEY" -H "Content-Type: application/json" \
  -d '{"model":"gemini-2.5-flash","messages":[{"role":"user","content":"Hi"}]}'

Free models: gemini-2.5-flash, gemini-2.0-flash, gemini-2.0-flash-lite.

Pool Gemini with other free tiers

The ~1,500/day Flash allowance is real but limited. freellmpool pools Gemini with Groq, Cerebras, NVIDIA and others so you draw on all of them and fail over automatically:

pip install freellmpool
export GEMINI_API_KEY=...        # plus other free keys
freellmpool ask -p gemini "Summarize this" < notes.txt   # pin Gemini
freellmpool ask "..."                                     # or pool + fail over

freellmpool handles Gemini's distinct request shape for you, so it works through the same OpenAI-compatible interface as the rest. See using multiple free LLM APIs together.

FAQ

Is the Gemini API free?

Yes. Google AI Studio provides a free Gemini API tier. Create a key at aistudio.google.com/apikey; the free tier is rate-limited (around ~1,500 requests/day on Flash models; verify current limits).

Which Gemini models are free?

Free Gemini models include Gemini 2.5 Flash, 2.0 Flash, and 2.0 Flash-Lite — the Flash family is the free tier's workhorse.
