Free Mistral API

Mistral offers a free experimentation tier. Get a free key at console.mistral.ai/api-keys; the endpoint is OpenAI-compatible at https://api.mistral.ai/v1, and free models include Mistral Small, Mistral Nemo, and Codestral (strong at code). To combine Mistral's free quota with other providers and fail over when it's rate-limited, pool it with freellmpool.

Use Mistral's free API directly

curl https://api.mistral.ai/v1/chat/completions \
  -H "Authorization: Bearer $MISTRAL_API_KEY" -H "Content-Type: application/json" \
  -d '{"model":"mistral-small-latest","messages":[{"role":"user","content":"Hi"}]}'

Free models include mistral-small-latest, open-mistral-nemo, and codestral-latest (code-specialized).

Pool Mistral with other free tiers

Codestral is a nice free option for code tasks; pair it with faster general models via freellmpool:

pip install freellmpool
export MISTRAL_API_KEY=...        # plus other free keys
freellmpool ask -m codestral-latest "Write a Python function to dedupe a list"
freellmpool ask "..."            # pool + fail over

FAQ

Is the Mistral API free?

Yes, Mistral has a free experimentation tier. Create a key at console.mistral.ai/api-keys and call the OpenAI-compatible endpoint at api.mistral.ai/v1. Free usage is rate-limited; verify current limits.

Is Codestral free to use?

Codestral, Mistral's code model, is available on the free tier via codestral-latest. It's a good free choice for code generation and completion; pool it with other free models using freellmpool.
