Free Cerebras API

Cerebras offers a free inference API with one of the largest daily request caps among free tiers (roughly ~14,400 requests/day; verify current limits) and very fast generation. Get a free key at cloud.cerebras.ai; the endpoint is OpenAI-compatible at https://api.cerebras.ai/v1 with free models like gpt-oss-120b and GLM 4.7. Pool it with other free tiers using freellmpool for failover when you hit a limit.

Use Cerebras' free API directly

curl https://api.cerebras.ai/v1/chat/completions \
  -H "Authorization: Bearer $CEREBRAS_API_KEY" -H "Content-Type: application/json" \
  -d '{"model":"gpt-oss-120b","messages":[{"role":"user","content":"Hi"}]}'

Free models: gpt-oss-120b, zai-glm-4.7.

Pool Cerebras with other free tiers

Cerebras' high daily cap makes it a great primary, with others as backup. freellmpool routes to Cerebras first when you prefer it and fails over to Groq, Gemini, NVIDIA and the rest when needed:

pip install freellmpool
export CEREBRAS_API_KEY=...       # plus other free keys
freellmpool ask -p cerebras "Refactor this function" < code.py
freellmpool benchmark             # time each provider so routing prefers the fastest

FAQ

Is the Cerebras API free?

Yes. Cerebras has a free inference tier with a comparatively large daily request cap (around ~14,400 requests/day; verify current limits). Get a key at cloud.cerebras.ai and call the OpenAI-compatible endpoint at api.cerebras.ai/v1.

What's special about Cerebras' free tier?

Two things: a large daily request allowance relative to other free tiers, and very fast token generation. It's a strong default to pool as a primary provider with freellmpool.
